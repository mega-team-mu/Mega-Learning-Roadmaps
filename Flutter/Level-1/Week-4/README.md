### Task 1:

Please, answer the following questions :

- **Proplem 1:**
    - Create a class called `Book` with:
        - Attributes: `title`, `author`, `isAvailable`
        - Methods:
            - `borrow()` ‚Üí makes the book unavailable.
            - `returnBook()` ‚Üí makes the book available again.
    - Create an object of a book and test borrowing and returning it.
- **Proplem 2:**
    - Create a class `Person` with:
        - Attributes: `name`, `age`
        - Method: `sayHello()` ‚Üí prints "Hello, my name is [name]"
- **Proplem 3:**
    
    Create a class `Rectangle` with:
    
    - Attributes: `width`, `height`
    - Method: `getArea()` ‚Üí returns area of the rectangle.
- **Proplem 4:**
    - Create a class called `Item` with:
        - Attributes: `name`, `price`
    - Create another class called `Cart` with:
        - Attribute: `items` (list of items)
        - Methods:
            - `addItem(item)`
            - `removeItem(item)`
            - `getTotal()` ‚Üí returns the sum of all item prices.
- **Proplem 5:**
    - Create a class called `BankAccount` with:
        - Attribute: `balance`
        - Methods:
            - `deposit(amount)` ‚Üí increases the balance.
            - `withdraw(amount)` ‚Üí decreases the balance if there is enough money.
            - `getBalance()` ‚Üí returns the current balance.
    - Try creating an account, deposit money, and withdraw some.
