## ğŸ“Š Use Case Diagram Elements

| Element | Symbol | Description |
| --------- | -------- | ------------- |
| **Actor** | Stick figure | External entity interacting with system |
| **Use Case** | Oval | System functionality |
| **Association** | Solid line | Actor-use case relationship |
| **Include** | Dashed arrow (`<<include>>`) | Required functionality |
| **Extend** | Dashed arrow (`<<extend>>`) | Optional functionality |
| **System Boundary** | Rectangle | Scope of the system |
