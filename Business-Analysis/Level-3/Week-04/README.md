## üìä AS-IS vs TO-BE Mapping

| Aspect | AS-IS | TO-BE |
| -------- | ------- | ------- |
| **Purpose** | Document current state | Design future state |
| **Focus** | Problems and bottlenecks | Solutions and efficiency |
| **Stakeholders** | Process owners | Change champions |
| **Output** | Gap identification | Improvement roadmap |

---

## üîß Six Sigma DMAIC

| Phase | Focus | BA Activities |
| ------- | ------- | --------------- |
| **D**efine | Problem scope | Requirements gathering |
| **M**easure | Current performance | Data collection |
| **A**nalyze | Root causes | Process analysis |
| **I**mprove | Solutions | Solution design |
| **C**ontrol | Sustain gains | Monitoring setup |

## ‚ôªÔ∏è Types of Waste (Lean)

| Type | Description | Example |
| ------ | ------------- | --------- |
| **Overproduction** | Making too much | Unused reports |
| **Waiting** | Idle time | Approval delays |
| **Transport** | Unnecessary movement | Data transfers |
| **Overprocessing** | Doing more than needed | Redundant reviews |
| **Inventory** | Excess stock | Backlog bloat |
| **Motion** | Unnecessary actions | Multiple logins |
