# Frontend Development Roadmap

This roadmap is designed specifically for Mega Team members to guide their learning journey in frontend development over the course of 12 months. The program is structured into three levels: Beginner, Mid-level, and Advanced. Each level includes learning objectives, tools, and practical projects to strengthen both theoretical and hands-on skills.

By the end of this roadmap, Mega Team members will have the skills and portfolio required to pursue professional opportunities in frontend development.

## Overview

**Total Duration:** 12 Months  

- Beginner: 3 months  
- Mid-level: 4 months  
- Advanced: 5 months  

This roadmap emphasizes progressive learning, starting from foundational web technologies and advancing to production-ready full-scale applications. It incorporates Arabic-language resources from Elzero Web School and other sources for accessibility. Focus on daily study (1-2 hours), weekly reviews, and building a GitHub portfolio.

Divided into 16 weeks, focusing on JavaScript, React, and related tools.

| Week | Learning Content | Tasks/Exercises |

[Playlist](https://www.youtube.com/playlist?list=PLDoPjvoNmBAx3kiplQR_oeDqLDBUDYwVv) | Create a simple app like a calculator with JS. Experiment with loops on arrays. |
| 4-6 (JS - Weeks 4-6/6) | #057 to #188 (Functions, scope, higher-order functions, objects, DOM, BOM, destructuring, sets/maps, regex, OOP, JSON, async/promises, fetch). [Playlist](https://www.youtube.com/playlist?list=PLDoPjvoNmBAx3kiplQR_oeDqLDBUDYwVv) | Build form validation with regex. Add events to a previous HTML/CSS page. |
| 7-9 (React - Weeks 1-3/6) | Arabic React Course - First 7 videos (Hello-world, components, state/props, events, conditional rendering, lists, hooks). [Playlist](https://www.youtube.com/playlist?list=PLQtNtS-WfRa9LbmD8ON7rWhn-AtKTGdkn) | Create a counter app with useState. Add list rendering. |
| 10-12 (React - Weeks 4-6/6) | Remaining videos (useEffect, e-commerce app, CRUD, state lifting, Redux basics, cart app, deploy). [Playlist](https://www.youtube.com/playlist?list=PLQtNtS-WfRa9LbmD8ON7rWhn-AtKTGdkn) | Build a basic e-commerce or CRUD app. Use Context API. |
| 13 (State Management - Week 1/2) | From React playlist: State Lifting Up, Props Drilling, Context API, Recoil, Using Redux, Build Redux, React-Redux, Toolkit. | 14 (State Management - Week 2/2) | Redux-related from React. [Playlist](https://www.youtube.com/playlist?list=PLQtNtS-WfRa9LbmD8ON7rWhn-AtKTGdkn) | Complete cart app with Redux Toolkit. Try dev tools. |
| 15 (APIs & Axios - Week 1/2) | Axios Course - All 6 videos (Get, post, delete, instance, errors). + Review JS async parts. [Playlist](https://www.youtube.com/playlist?list=PLxRvAw0S16C58zrvZtvAonjdR-lvnFIh_) | Fetch data from a public API (e.g., JSONPlaceholder) with Axios. |

## Advanced Level (5 Months)

**Objective:** Master production-ready tools and build full-scale applications.

Divided into 20 weeks, emphasizing TypeScript, Next.js, and optimization.

| Week | Learning Content | Tasks/Exercises |
| --- | --- | --- |
| 1-2 (TypeScript - Weeks 1-2/3) | Learn Typescript 2022 - #01 to #25 (Introduction, types, functions, aliases, interfaces). [Playlist](https://www.youtube.com/playlist?list=PLDoPjvoNmBAy532K9M_fjiAmrJ0gkCyLJ) | Convert previous JS code to TS. Add types to functions. |
| 3 (TypeScript - Week 3/3) | #26 to #38 (Classes, generics, mastering). [Playlist](https://www.youtube.com/playlist?list=PLDoPjvoNmBAy532K9M_fjiAmrJ0gkCyLJ) | Build a class-based app with TS. Experiment with generics. |
| 4-7 (Next.js - Weeks 1-4/7) | Next.js 15 Tutorial - #1 to #50 (Introduction, SSR/SSG, routing, API routes). + Next 13 video. [Playlist](https://www.youtube.com/playlist?list=PLC3y8-rFHvwhIEc4I4YsRz5C7GOBnxSJY) | Create a basic Next app with TS. Add pages and API. |
| 8-10 (Next.js - Weeks 5-7/7) | #51 to #92 (Auth, settings, RBAC, deploy). [Playlist](https://www.youtube.com/playlist?list=PLC3y8-rFHvwhIEc4I4YsRz5C7GOBnxSJY) | Integrate auth into an app. Protect routes. |
| 11 (Auth - Week 1/2) | From Next.js: Auth-related (#80s-90s), JWT, OAuth, Firebase/Auth0. (Check resources) | Add login/signup with Clerk or Auth0 to a project. |
| 12 (Auth - Week 2/2) | Complete auth, role-based access. (Check resources) | Experiment with OAuth in a real app. |
| 13 (Testing - Week 1/2) | Learn Jest - #01 to #10 (Introduction, tests, matchers). (Check resources) | Write unit tests for functions in TS. |
| 14 (Testing - Week 2/2) | #11 to #20 (Advanced matchers, mocks, coverage). (Check resources) | Test React components with RTL. Aim for 80% coverage. |
| 15 (Perf Opt - Week 1/2) | From Next.js: Perf-related (lazy loading, memo, splitting). (Check resources) | Optimize an app: Add lazy loading, measure performance. |
| 16 (Perf Opt - Week 2/2) | Complete, trainings. (Check resources) | Rebuild an app with code splitting. |
| 17 (Deployment - Week 1/2) | From Next.js: Deploy to Vercel/Netlify, Docker basics. (Check resources) | Deploy an app to Vercel. Try Docker container. |
| 18 (Deployment - Week 2/2) | Complete, cloud basics. (Check resources) | Add CI/CD with GitHub Actions. |
| 19-20 (Final Projects - Weeks 1-2/4) | Build E-commerce app (use Next + TS + Redux). (Check resources) | Build features like cart, auth. Deploy. |
| 21-22 (Final Projects - Weeks 3-4/4) | Social media clone + real-time chat (use Firebase for real-time). (Check resources) | Integrate APIs, test, add to portfolio. Full review. |
